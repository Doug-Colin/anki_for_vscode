## What's an event listener?

An event listener is a callback function that JS registers with the host environment for invoking upon triggering events.

[#callbacks]() [#events]() [#DOM]()

<!------------------------------------------------------------------------------------------- -->

## What's an event handler?

An event handler is a callback function that JS registers with the host environment for invoking upon triggering events.

[#callbacks]()

<!------------------------------------------------------------------------------------------- -->

## What types of events trigger event handlers in JS?

Event handler callbacks are triggered by user interactions like click, hover, or submit and system or network events like timers, I/O completion, and other system signals.
