## How do you format code blocks and inline code in markdown?

Code block: wrap in triple backticks ` `
Inline: single backticks ` `

## How do you format text to be italic in markdown?

%
By surrounding it with the asterisk `*` or underscore `_` characters.

Example:
`Markdown renders any text between *aserisks* or _underscores_ as italic`
will render as:
Markdown renders any text between _aserisks_ or _underscores_ as italic
